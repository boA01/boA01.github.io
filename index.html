rpm命令
rpm -q 查询指定的包是否安装
rpm -ql 查询包生成的所有文件

rpm -q –changelog *.rpm 查询包更新信息
rpm -ivh *.rpm 安装
rpm -ivh –force *.rpm 强制安装
rpm -ivh –nodeps –force *.rpm 忽略依赖关系
rpm -ivh –replacepkgs *.rpm 覆盖、替换安装
rpm -e *.rpm 卸载
rpm -e –nodeps *.rpm 忽略依赖关系
rpm -e –noscripts *.rpm 不执行脚本卸载
rpm -Uvh *.rpm 升级
-U和-F选项均可以升级一个包。旧包不存在时，
-U选项会安装新包，而-F选项不安装新包

apt命令
sudo apt-get update # 更新本地包数据库
sudo apt-get upgrade # 更新所有已安装的包
sudo apt-get install # 安装
sudo apt-get autoremove # 自动移除不需要的包
补充几个常用的清理命令：
apt-get autoclean: 将已删除软件包的.deb安装文件从硬盘中删除；
apt-get clean: 同上，但会把已安装的软件包的安装包也删除掉；
apt-get autoremove: 删除为了满足其他软件包的依赖而安装，但现在不再需要的软件包；
apt-get remove : 删除已安装的软件包（保留配置文件）；
apt-get –purge remove [软件包名]: 删除已安装包（不保留配置文件）
sudo apt-cache search package_name # 搜索软件
sudo apt-cache show package_name # 查看软件包信息
sudo apt-cache show depends package_name # 查看软件包依赖关系

5.文件
权限
基本权限: rwxrwxrwx, 421421421
chgrp
chown [-R] user:grap
chmod [-R] 777 :

隐藏属性: a,i
chattr [+-=][ASacdistu]:设定
a: 档案将只能增加数据，而不能删除也不能修改数据，只有root可设定
c: 自动将此档案压缩，读取时候自动解压缩
i: 档案不能被删除、改名、设定连结也无法写入资料，只有root能设定
s: 档案被删除，他将会被完全移除出这个硬盘,如果误删了，完全无法救回
u: 与s相反，可以用来救援该档案
lsattr [-adR]: 查看

特殊权限：sxt，421(S,T代表x为空)
SUID: s 执行者将具有该程序拥有者的权限(例如passwd); 对程序有效
SGID: x 执行者在执行过程中将会获得该程序群组的支持; 对档案,目录有效
SBIT: t 用户在该目录下建立档案或目录时，仅自己与root有权力删除该档案;对目录有效

预设权限: -rw-rw-rw-, 666; drwxrwxrwx, 777
umask 022/[-S]: 预设(644)(755)/查看

    ctrl rn: 后退n步

    ctrl x：
    ctrl a：

命令模式：
    a: 从光标所在的下一个字符处开始插入
    A: 从光标所在行的最后一个字符处开始插入
    i: 从光标所在处插入
    I: 在所在行的第一个非空格符处开始插入
    o: 在光标所在的下一行插入新的一行
    O: 在光标所在的上一行插入新的一行
    r: 取代一个字符
    R: 一直取代，直到按下ESC为止
    s: 剪切当前字符开始插入
    S: 剪切当前行开始插入
    C: 剪切到行尾开始插入
    cc: 剪切当前行开始插入
    ctrl c: 剪切光标后开始插入
    ncj: 如向下删除n行开始插入
    cw：更改光标所在处的字
    cnw：更改n个字
    ZZ: 保存退出

    v: 字符选择
    V: 行选择
    Ctrl v: 块选择
    y: 将反白的地方复制起来
    d: 将反白的地方删除掉

    yw: 复制一个单词
    [n]yy: [n]行复制
    [n]x: 向后剪切一[n]个字符，(cut)
    [n]X: 向前剪切一[n]个字符
    p: 粘贴到光标下一行
    P: 粘贴在游标上一行
    dw: 删除一个单词(delets)
    D: 删到行尾
    [n]dd: 删[n]行
    dG: 删到文尾
    d[n]gg: 删到文首,删n行
    ctrl g：显示行数百分比
    ctrl r: 重做上一个动作(redo)
    u: 复原前一个动作(undo)
    ctrl z: 撤销
    .: 重复操作

    /, ?word：向光标之上，下查找word
    n: 重复正向查找操作
    N: 重复反向查找操作

    [n]h, ← 左移n
    [n]j, ↓, + 下移n
    [n]k, ↑, - 上移n
    [n]l, → 右移n
    [n] space: 右移n

    b: 上一个单词头
    w: 下一个单词头
    e: 下一个单词尾
    ff: 光标找到本行下一个f

    H: 光标移到屏幕最上方那一行的第一个字符

    ctrl d: 下移半页
    ctrl f: 相当于PgUp
    ctrl b: 相当于PgDn
    ctrl p: 下一行
    ctrl n,m: 下一行
    [n]gg: 首行，第n行
    G: 末行
    ctrl t: 
    J: 将光标所在列不下一列的数据结合成同一列
    缩徘: <<,>>
    段数切换：{/}
    括号匹配及切换：%

末行模式：
    :q! #强制退出
    :[n1,n2]w [f_n] #[n1到n2行]保存[到f_n]
    :wq! , :x , ZZ #三者相同，保存退出
    :[v]ps 文件路径 #分屏; ctrl w j/k 换页
    :n [f_n] #编辑下一个档案, [新建f_n]

    :n1,n2s/o/n/g[c]  #n1到n2行[:1,$s 全局]的o变n；c确定提示
