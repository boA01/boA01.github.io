mysql

创建数据库：create database base_name;
查看数据库：show database;
使用数据库：use base_name;
查看数据表：show tables;
创建数据表: create table table_name(key type 约束);[例如：(name varchar(25),id int)]
插入数据：insert into table_name(id,name,sex) values (001,'lizhaobo','male');
查看指定数据表：desc table_name;

约束：NOT NULL(不为空); UNIQUE(唯一); PRIMARY KEY (学号, 科目)(主键); FOREIGN KEY (Id)(参照完整性); CHECK(条件); DEFAULT(设置默认)

vscode下mysql建立数据库的方法：

先在终端下载安装Python3-pip;
安装mysql包：pip3 install pymysql;
安装mysql插件：MySQL management tool、MySQL client、SQLtools
打开MySQL，添加MySQL服务；
在mysql-shell下创建数据库，然后在可视化下查看数据表。
