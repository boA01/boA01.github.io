安装系统

Posted on 2020-09-05
Symbols count in article: 476 Reading time ≈ 1 mins.

可用电脑一台，8G左右u盘一个 (^干净的)
[^干净的]:没被玷污的！！！

*一.操作系统下载
官方网站选择不同版本的镜像

*二.下载启动盘制作工具
如UitraISO,banlenaEtcher等

*三.制作启动盘
1.打开UltraISO
2.继续试用
3.文件->打开->选择镜像文件->打开->启动->写入硬盘映象->写入
注意：硬盘驱动器选择u盘

*四.安装操作系统
1.插上u盘
2.进入BIOS（不同电脑不一样，自己百度，华硕主板F2）
3.选择启动项，把u盘拖到第一个
4.保存退出（F7）
5.接下来就是鼠标操作，识字就行

小白到此结束（操作系统自动安装驱动程序）!!!

*五.手动安装驱动程序

1主板的驱动程序
华硕：https://www.asus.com.cn/support/Download-Center/

2独显的驱动程序
N卡：https://www.nvidia.cn/Download/index.aspx?lang=cn
A卡：https://www.amd.com/zh-hans/support

3其他设备驱动程序
自己查找
