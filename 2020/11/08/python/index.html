虚拟环境

创建ll-env虚拟环境

`virtualenv ll_env` `python3 -m venv ll_env` `pipenv install [--python 3.8] [产生Pipfile和pipflie.lock文件]按Pipfile文件生成环境(安装包)` `conda create -n [,....] [-y]`

进入虚拟环境

`W:ll_env\Scripts\activate` `L:[source] ll_env/bin/activate`

pipenv shell 没有环境就自动初始化

[source] activate <env_name>

退出虚拟环境

`[source] deactivate`
