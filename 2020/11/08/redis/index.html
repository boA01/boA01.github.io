hash

设置键(h_k)值(kn vn)
hset h_k k1 v1 k2 v2 …

获取值
hget h_k kx
hmget h_k k1 k2 …
hgetall h_k

值运算
kincrby h_k kx int(10,-1)

hkeys h_k

list

创建键(l_k)值(vn(可重复))
r/lpush l_k v1 v2 v2 …

l/rset l_k index n v
lindex l_k 1

查看所有值
lrange l_k 0 -1

所有值的数目
llen l_k

删除零号位值
l/rpop l_k

lrem l_k n v

创建键(s_k)值(vn)
sadd s_k v1 v2 …

查看所有值
smembers s_k

查看是否存在v
sismember s_k v

随机返回n个元素
srandmember s_k n

zset(按score_double排序)

zadd zs_k 456 v1 478 v2 478 v3

zincrby zs_k +/-300 vx

查看所有元素
zrange zs_k 0 -1 [withscores]

查看kx的sd
zscore zs_k kx

查看min-max的元素数目
zcount zs_k min max

返回min-max分段的第offset位后count位元素
zrangebyscore zs_k min max withscore [limit offset count]

zrem zs_k km [kn ..]
zremrangebyscore zs_k min max

import redis as rd

client = rd.Redis(host=’localhost’, port = 6379, db = 0)

print(client.keys())
client.set(‘book5’,’redis’)
client.append(‘book5’, ‘-client’)
print(client.get(‘book5’).decode())
client.lpush(‘zb’, 18, ‘cs’, ‘play’)
print(client.lrange(‘zb’, 0, 2))
client.sadd(‘hello’, ‘zb’, ‘sx’)
print(client.smembers(‘hello’))

all_info = client.keys
for key in all_info():
print(key.decode())
