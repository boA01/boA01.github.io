hash

设置键(h_k)值(kn vn)
hset h_k k1 v1 k2 v2 …

获取值
hget h_k kx
hmget h_k k1 k2 …
hgetall h_k

值运算

list

创建键(l_k)值(vn(可重复))
r/lpush l_k v1 v2 v2 …

l/rset l_k index n v
lindex l_k 1

查看所有值
lrange l_k 0 -1

所有值的数目
llen l_k

删除零号位值
l/rpop l_k

lrem l_k n v

sadd s_k v1 v2 …

查看所有值
smembers s_k

sismember s_k v

srandmember s_k n

差集运算

sunionstore d s_km s_kn …(结果放入d)

zscore zs_k kx

zrangebyscore zs_k min max withscore [limit offset count]
